# Guide de gestion des annonces spéciales

## Comment utiliser les annonces

Les annonces spéciales vous permettent d'afficher des messages importants sur votre page d'accueil pour des événements, vacances, ou occasions spéciales.

### Accéder à la section Annonces

1. Allez sur votre interface d'administration : `https://votre-site.com/admin`
2. Connectez-vous avec vos identifiants
3. Dans le menu latéral, cliquez sur **"⚙️ Paramètres"**
4. Cliquez sur **"Annonce spéciale"**

### Configuration de l'annonce

Vous pouvez configurer les éléments suivants :

#### 1. **Afficher l'annonce** (Case à cocher)
- Cochez cette case pour afficher l'annonce sur votre site
- Décochez-la pour la masquer
- L'annonce n'apparaîtra que si cette case est cochée

#### 2. **Titre (FR)** et **Titre (EN)**
- Saisissez le titre de votre annonce en français et en anglais
- Exemples : "Fermeture exceptionnelle", "Menu spécial Noël", "Réouverture"

#### 3. **Description (FR)** et **Description (EN)**
- Rédigez le texte complet de votre annonce
- Vous pouvez utiliser plusieurs lignes
- Soyez clair et concis

#### 4. **Image** (Optionnel)
- Ajoutez une image illustrative pour votre annonce
- Cliquez sur "Choisir une image" pour télécharger une image depuis votre ordinateur
- L'image apparaîtra à gauche de votre texte sur grand écran
- Si vous ne mettez pas d'image, seul le texte sera affiché

#### 5. **Couleur d'arrière-plan**
- Personnalisez la couleur d'arrière-plan de la bannière
- Utilisez un code couleur hexadécimal (ex: #FEF3C7 pour jaune pâle)
- Par défaut : `#FEF3C7` (jaune pâle)
- Autres exemples :
  - `#DBEAFE` - Bleu pâle
  - `#FEE2E2` - Rouge pâle
  - `#D1FAE5` - Vert pâle

### Exemples d'utilisation

#### Fermeture exceptionnelle
```
Afficher l'annonce : ✓ Coché
Titre (FR) : Fermeture exceptionnelle
Titre (EN) : Exceptional closure
Description (FR) : Le restaurant sera fermé du 24 au 31 décembre pour les fêtes de fin d'année. Nous vous souhaitons de joyeuses fêtes !
Description (EN) : The restaurant will be closed from December 24 to 31 for the holidays. We wish you happy holidays!
Image : (optionnel)
Couleur d'arrière-plan : #FEE2E2
```

#### Menu spécial
```
Afficher l'annonce : ✓ Coché
Titre (FR) : Menu spécial Saint-Valentin
Titre (EN) : Valentine's Day Special Menu
Description (FR) : Découvrez notre menu spécial pour la Saint-Valentin ! Réservation recommandée au 02 33 92 18 45
Description (EN) : Discover our special Valentine's Day menu! Booking recommended at 02 33 92 18 45
Image : (photo du plat spécial)
Couleur d'arrière-plan : #FEF3C7
```

#### Réouverture
```
Afficher l'annonce : ✓ Coché
Titre (FR) : Nous sommes de retour !
Titre (EN) : We're back!
Description (FR) : Le restaurant rouvre ses portes le 15 janvier. Nous avons hâte de vous accueillir à nouveau !
Description (EN) : The restaurant reopens on January 15th. We look forward to welcoming you again!
Image : (photo de la façade)
Couleur d'arrière-plan : #D1FAE5
```

### Publier ou masquer l'annonce

- Pour **publier** l'annonce : Cochez la case "Afficher l'annonce" et cliquez sur "Enregistrer"
- Pour **masquer** l'annonce : Décochez la case "Afficher l'annonce" et cliquez sur "Enregistrer"

L'annonce apparaîtra automatiquement sur votre page d'accueil, juste en dessous de l'image d'en-tête.

### Emplacement sur le site

L'annonce s'affiche :
- **Position** : Entre l'image d'en-tête (Hero) et la section "À propos"
- **Visibilité** : Visible sur toutes les tailles d'écran (ordinateur, tablette, mobile)
- **Design** : Carte blanche sur fond coloré, avec icône d'alerte

### Conseils

- Mettez à jour l'annonce régulièrement pour garder votre site dynamique
- N'oubliez pas de désactiver l'annonce une fois l'événement passé
- Utilisez des images de bonne qualité (format recommandé : JPG ou PNG, max 500 Ko)
- Rédigez des textes courts et percutants
